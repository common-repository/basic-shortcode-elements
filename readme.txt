A simple shortcodes plugin that allows you to include some of elements as: buttons, parallax image, two table styles, two styles of latest posts, etc

== Description ==

Provides several components such as: one or two columns, two table styles and different button colors, parallax image, last posts to show and share with differents social media like: Twitter, Facebook, Google+, Pinterest and Whatsapp.


== Installation ==

1. Upload the entire `basic_shortcode_elements` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.

You will find the button '[] Add Element Shortcode ' in the page edition , right next to the media button

For basic usage, you can have a look at http://amrani.es/wp-plugins/basic-shortcode-elements.php
